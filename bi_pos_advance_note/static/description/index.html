Point of Sales Advance Order Note on Order line and Order Item Odoo Apps

This Odoo Apps is useful for place such as restaurants where you need add Note for items particular product as well as for the whole Order in Point of sales for customize order or points of sales order also its printed on bill receipt. This odoo apps module also helps to create new order note and same note you can use on different POS orders which you can add on point of sales whole order, order lines as well as on each item of pos order lines, also it has feature to print those notes on pos receipt.

Features

Dynamically Create Notes.

User can create notes dynamically in Backend side and on POS Screen.

Multi Notes Selection & Multi Type Apply.

Easy to select multiple notes and easy to apply multi type like individual product and all order on POS Screen.

See All Notes.

User can see all created notes.

POS Order With Notes in Backend.

User can see selected notes in backend pos order.

Print Notes in Receipt.

User can see selected notes in printed POS Receipt.

User Configuration.

User can Add New Note and Edit Predefined Notes.
Demo Request Video Tutorial

Create or Edit Predefined Notes in Backend

POS Order Notes Menu.

Under the Configuration menu in Point of Sale, find the POS Order Note Menu for adding or edit the Predefined Notes.

Notes Button in POS Screen

Click on the Notes Button to select from Predefined Notes.

Notes Popup

In this Popup window Select Note type as "Product Note" or "Order Note". Here you can add Manual Notes as well as select from Predefined Notes also. Then Click "Apply".

Notes Added

Based upon the Note Type, Notes will be added for product or for the Whole Order.

Notes Added in Receipt

Based upon the Note Type, Notes will be added in POS Receipt for product or for the Whole Order.

In the back-end: Order

In POS Order Notes will be Added.
